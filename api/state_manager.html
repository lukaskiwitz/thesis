StateManager¶

class thesis.main.StateManager.StateManager(path, debug=False)¶

Bases: thesis.main.SimComponent.SimComponent

This class manages the state of the simulation. Its main use is to run parameter scans in an organised manner and produces the xml files necessary for the use of PostProcessor.
