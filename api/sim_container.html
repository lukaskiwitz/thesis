SimContainer¶

class thesis.main.SimContainer.SimContainer(parameter_set)¶

Bases: thesis.main.SimComponent.SimComponent

Container to hold all simulation object. Can be used as standalone or aggregate with StateManager.

Variables

  • p (Parameterset) – parmeters set, that get handed down to global problems.

  • entity_list (List[Entity]) – List of all entities in the simulation

  • global_problems (List[GlobalProblem]) – list of global problems in this simulation

  • top_path – path to the top level directory to serve as reference for ext_cache

  • path – (sub) path to simulation results

  • relative_path (str) –

  • worker_sub_path (str) –

  • entity_templates (List[EntityType]) – list of available entity templates

  • internal_solvers (List[InternalSolver]) – list of available internal solvers

  • marker_lookup (Mapping[str,int]) – lookup dict to manually set the mesh function value for exported properties

  • markers (List[str]) – list of entity properties, which should be exported as mesh functions

  • t (float) – current time

  • record (ClassRecord) – Class record object to store execution timing data

  • default_sample – scan sample to reset sim object before each parameter scan
