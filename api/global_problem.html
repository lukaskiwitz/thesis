GlobalProblem¶

class thesis.main.FieldProblem.GlobalProblem¶

Bases: abc.ABC

Abstract base class for all global problems. New global models should be implemented by subclassing from this class and implementing all its abstract methods.

Variables

  • path (st) – file path to result directory for this problem

  • name – nonunique string identifier

  • field_name (str) – string to identify this field

  • field_quantity (str) – physical properties associated with this field

  • registered_entities (List[Entity]) – entities that interact with this field

  • solver (MySolver) – solver to be used for this field

  • p (ParameterSet) – parameter dict to be handed down

FieldProblem¶

class thesis.main.FieldProblem.FieldProblem¶

Bases: thesis.main.FieldProblem.GlobalProblem

Sets up a mesh based boundary value problem from registered entities and outer domain attributes,

to be solved with a given solver. This class is meant to aggregate with SimContainer to automate the simulation pipeline.

Variables

  • path – file path to result directory for this problem

  • ext_cache (str) – path to folder containing pre generated mesh

  • outer_domain (DomainEntity) – Domain Entity which represents the simulation domain

  • solver (MyDiffusionSolver) – solver to be used for this field

  • remesh_scan_sample (bool) – flag to trigger remeshing at the beginning of a parameter scan

  • remesh_timestep (bool) – flag to trigger remeshing before each timestep

  • moving_mesh (bool) – flag that indicates that the mesh is change during simulation
